# Tudengibaar

**Autor:** Sander Pihelgas

**Programmi sisu:** Programm annab kasutajale ülevaate võimalikest valmistatavatest kokteilidest vastavalt olemasolevatele koostisosadele

**Kasutusjuhend:** 
Programmi avaaknas kuvatakse 4 nuppu. 
Vasakus veerus on olemasolevate jookide nupud, vastavalt Add Inventori ja View Inventori. 
Paremas veerus on retseptinupud, vastavalt Add Receipe ja View Receipe. 

Programmi kasutamist tuleks alustada olemasolevate koostisosade lisamisega (kogused esitada liitrites ja eraldada punktiga)

"Add Inventori" lisab olemasolevad joogid
"View Inventori" kuvab olemasolevad koostisosad
"Add Receipe" võimaldab lisada retsepte
"View Receipe" kuvab koostise suhtarvud, kokteili kanguse ning arvutab mitu 250 ml suurust kokteili on võimalik nendest koostisosadest valmistada
